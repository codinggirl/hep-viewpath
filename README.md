# hep-viewpath

[![Greenkeeper badge](https://badges.greenkeeper.io/codinggirl/hep-viewpath.svg)](https://greenkeeper.io/)

A view path middleware for Hep/Koa.

## 安装

- 运行 npm 命令行安装

```
npm i -S hep-viewpath
```

- 运行 cnpm 命令行安装

```
cnpm i -S hep-viewpath
```
